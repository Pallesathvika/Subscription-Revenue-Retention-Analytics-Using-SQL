# 📊 Subscription Revenue & Retention Analytics (SQL Project)

## 📌 Project Overview
This project analyzes subscription-based business data using MySQL.  
The objective is to calculate key business KPIs such as Monthly Revenue, Revenue Growth, Customer Lifetime Value (CLTV), and Rolling Averages using advanced SQL concepts.

This project demonstrates strong SQL fundamentals along with CTEs and Window Functions.

---

## 🛠 Tools Used
- MySQL
- MySQL Workbench
- GitHub

---

## 🗂 Database Schema

The database consists of four tables:

1. customers
2. plans
3. subscriptions
4. payments

Relationships:
- One customer can have multiple subscriptions.
- One plan can be subscribed to by multiple customers.
- One subscription can have multiple payments.
